## 🧠 Overview

MECKARUP is a **production-ready service + education platform** designed to bridge the gap between **learning and real-world execution**.

It enables users to:

- 📊 Book research & consultation services  
- 💻 Request custom web development solutions  
- 📈 Get data analytics & statistical support  
- 🎬 Access VFX & design services  
- 🎯 Prepare for placements with guided assistance  

The platform integrates **service delivery, skill development, and practical implementation** into a single ecosystem.

---

## 🚀 Problem & Solution

### ❌ Problem
Students, startups, and small businesses often lack access to affordable, practical, and real-world technical solutions in research, development, and analytics.

### ✅ Solution
MECKARUP provides a unified platform that combines **services, education, and execution**, enabling users to solve real problems with expert guidance and scalable solutions.

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- Custom CSS (fully responsive, no frameworks)
- React Router (SPA navigation)

### Backend (Serverless)
- Google Apps Script (API layer)
- Google Sheets (data storage)

### Deployment & Infrastructure
- Vercel (hosting & CI/CD)
- Cloudflare (DNS + SSL)

---

## 🌐 Core Features

### Service & Course System
- Dynamic rendering using structured data
- Dedicated detail pages for each service and course
- Modular and reusable component architecture

### Booking System (Core Functionality)

```
User → Select Service → Fill Form → Submit
↓
Apps Script API → Google Sheet → Success Modal
```

- Modal-based booking interface  
- Automatic service selection  
- Real-time data storage
